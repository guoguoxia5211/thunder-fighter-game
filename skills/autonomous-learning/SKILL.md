# 自主学习技能 (Autonomous Learning)

**版本：** 1.0  
**创建：** 2026-03-10  
**目标：** 从错误中自动总结，形成可复用规则

---

## 🎯 核心原则

**不重复犯同样的错误。每次错误都是学习机会。**

---

## 📋 学习流程

```
犯错/发现问题 → 记录 corrections.md
     ↓
分析根本原因 → 写清楚为什么错
     ↓
制定避免策略 → 具体可执行的规则
     ↓
同类错误≥3 次 → 提升为全局规则
     ↓
更新 MEMORY.md → 形成长期记忆
     ↓
更新 AGENTS.md → 改变工作流
```

## 📝 错误记录格式

```markdown
# 错误记录：[简短描述]

**日期：** YYYY-MM-DD
**严重性：** 高/中/低
**类别：** [VBR/决策/理解/执行/...]

## 发生了什么
[客观描述错误]

## 为什么错了
[根本原因分析]

## 如何修复的
[实际采取的行动]

## 如何避免再犯
[具体可执行的规则]

## 是否形成模式
- 同类错误次数：N 次
- 是否提升为规则：是/否
```

---

## 🔧 模式识别

当同类错误发生 3 次时：

1. **确认模式存在**
   - 读取 corrections/ 下所有记录
   - 统计相似错误次数
   - 确认是系统性问题

2. **创建全局规则**
   ```markdown
   # 规则：[规则名称]
   
   **来源：** 3 次同类错误总结
   **日期：** YYYY-MM-DD
   
   ## 规则内容
   [清晰、可执行的规则]
   
   ## 触发条件
   [何时应用此规则]
   
   ## 违规后果
   [不遵守会导致什么问题]
   ```

3. **更新工作流**
   - 添加到 AGENTS.md
   - 添加到 MEMORY.md
   - 更新相关技能文档
