# UI Design Skill - 游戏 UI 美术设计系统

## 描述

快速生成可定制的游戏 UI 主界面，支持实时预览和配置。适用于游戏主菜单、启动界面的快速原型设计。

## 使用场景

- 游戏主菜单 UI 设计
- UI 风格快速原型验证
- 多主题 UI 方案对比
- 移动端 UI 适配测试

## 核心能力

### 1. 布局系统
- 4 种预设布局（紧凑/标准/宽敞/专注）
- 预览区宽度可调（25%-50%）
- 响应式移动端适配

### 2. 主题风格
- 🚀 科幻未来（蓝紫渐变 + 科技感）
- ⛩️ 日式和风（红黄渐变 + 传统元素）
- ☢️ 末日废土（棕黄渐变 + 粗糙质感）
- 🌸 清新卡通（粉紫渐变 + 圆润风格）

### 3. 标题特效
- 背景光晕强度调节
- 文字阴影深度调节
- 光晕背景开关
- 8 种字体预设（粗黑/书法/像素/优雅/现代/可爱/潦草/赛博朋克）
- 字重/字间距精细调节

### 4. 背景系统
- 8 种底纹预设（点阵/条纹/暗角/噪点/网格/发光/径向/无）
- 底纹强度调节
- 自定义渐变背景
- 动态星星特效（0-200 颗）

### 5. 按钮系统
- 4 个功能按钮独立定制
- 文字内容可编辑
- 渐变色自定义
- 圆角精细调节（0-20px）
- 点击涟漪特效
- 高亮边框动画

### 6. 工具功能
- 网格线辅助（20px 网格）
- 面板全部展开/收起
- 自动保存（500ms 防抖）
- 设计导出（JSON 格式）
- 一键重置

## 调用方式

```bash
# 生成默认 UI 设计工具
openclaw skill ui-design-skill

# 或直接打开 HTML 文件
openclaw preview /root/.openclaw/workspace/skills/ui-design-skill/ui-design.html
```

## 技术特性

- **零依赖**：单 HTML 文件，无需构建
- **移动端优先**：触摸优化，响应式布局
- **实时预览**：配置即生效，无延迟
- **自动保存**：防止配置丢失
- **可导出**：JSON 格式保存设计方案

## 自定义扩展

### 添加新主题
在 `applyUIStyle()` 函数中添加新主题配置：

```javascript
'newtheme': {
    bg: 'linear-gradient(...)',
    btnStart: 'linear-gradient(...)',
    logoGradient: 'linear-gradient(...)',
    // ...
}
```

### 添加新字体
在 `selectFont()` 的 `fontConfig` 对象中添加：

```javascript
'新字体名': { 
    family: "'Font Name', sans-serif", 
    size: '40px', weight: '700', 
    textShadow: '...', letterSpacing: '2px'
}
```

### 添加新底纹
在 `applyTexture()` 的 `textures` 对象中添加：

```javascript
newtype: 'css-background-value'
```

## 最佳实践

1. **快速迭代**：先用预设主题，再微调
2. **移动端测试**：在设计阶段就测试移动端效果
3. **导出备份**：重要设计方案及时导出 JSON
4. **网格辅助**：精细调整时开启网格线

## 注意事项

- 字体依赖系统字体，跨平台可能略有差异
- 导出 JSON 仅保存配置，不包含完整 HTML
- 自动保存为本地会话级，刷新页面会重置
