## 设计技巧

### 1. 快速切换主题
点击右侧配置面板的"界面风格"，4 个主题一键切换，快速对比效果。

### 2. 精细调整
- 开启网格线（右下角📐按钮）辅助对齐
- 使用滑块微调数值（光晕/阴影/间距等）
- 展开所有面板（右下角📋按钮）查看全部配置

### 3. 按钮定制
每个按钮独立配置：
- 开始按钮：使用醒目的渐变色（蓝/绿/红）
- 皮肤按钮：暖色调（橙/粉）
- 成就按钮：紫色系（高贵感）
- 设置按钮：低饱和度（不抢眼）

### 4. 标题设计
- 字体选择决定整体风格
- 字间距影响视觉张力（2-6px 推荐）
- 光晕和阴影配合使用，增强立体感

### 5. 背景层次
- 底层：渐变背景色
- 中层：底纹纹理（30% 透明度）
- 上层：星星特效（动态闪烁）
- 顶层：网格辅助线（设计时开启）

## 常见问题

### Q: 字体显示不正确？
A: 字体依赖系统字体，跨平台可能略有差异。建议使用通用字体族（如 'Arial Black', 'Microsoft YaHei', sans-serif）。

### Q: 配置刷新后丢失？
A: 自动保存为会话级，刷新页面会重置。重要设计请及时导出 JSON。

### Q: 如何添加新按钮？
A: 在 HTML 的 `.menu-section` 中添加按钮元素，复制现有按钮结构，在 JS 中添加对应的配置面板和更新函数。

### Q: 如何修改网格大小？
A: 在 CSS 中修改 `.grid-overlay.active` 的 `background-size` 属性（默认 20px）。

## 进阶扩展

### 添加动画效果
在 CSS 中添加 `@keyframes` 动画，应用到按钮或标题元素。

### 添加音效
在 `handleClick()` 函数中添加音频播放逻辑。

### 多语言支持
将文字内容提取到配置对象，支持切换语言包。

### 主题市场
建立主题模板库，用户可上传/下载设计方案。
