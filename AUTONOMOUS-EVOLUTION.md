## 🔄 新工作流

### 代码开发流程 (VBR 强化)
```
旧流程：写代码 → 告诉用户"完成了" → 用户测试 → 发现 bug → 修复
新流程：写代码 → 自主验证 → 截图记录 → 修复问题 → 告诉用户"验证完成"
```

### 会话开始流程 (任务发现)
```
旧流程：等用户指示 → 执行任务
新流程：检查 Git 状态 → 扫描待办 → 主动汇报发现 → 执行高优先级任务
```

### 错误处理流程 (自主学习)
```
旧流程：犯错 → 修复 → 可能再犯
新流程：犯错 → 记录 corrections/ → 分析原因 → 制定规则 → ≥3 次提升为全局规则
```
