## 🎮 游戏设计

### 玩家坦克
```
- 移动速度：5 格/秒
- 生命值：3
- 子弹伤害：1
- 射击冷却：0.3 秒
- 最大子弹数：1（可升级）
```

### 敌方坦克 AI
```
状态机设计：
- 巡逻状态 → 随机移动
- 攻击状态 → 追踪玩家 + 射击
- 躲避状态 → 躲避玩家子弹
- 保护状态 → 回防基地

行为树优先级：
1. 看到玩家 → 攻击
2. 被子弹瞄准 → 躲避
3. 基地危险 → 回防
4. 默认 → 巡逻
```

### 地图元素
```
- 🧱 砖块：可被子弹摧毁
- 🟦 钢板：不可摧毁
- 🏠 基地：被摧毁游戏结束
- 🌊 河流：子弹可通过，坦克不可通过
- 🌲 树林：遮挡坦克（隐藏）
```

### 敌方坦克类型
```
| 类型 | 速度 | 生命 | 伤害 | AI 等级 |
|------|------|------|------|--------|
| 普通 | 中 | 1 | 1 | 基础 |
| 快速 | 快 | 1 | 1 | 追踪 |
| 重装 | 慢 | 3 | 1 | 防守 |
| 特种 | 中 | 2 | 2 | 智能 |
```

class AttackNode {
  execute(enemy, player) {
    if (enemy.canSeePlayer(player)) {
      enemy.aimAt(player);
      enemy.shoot();
      return SUCCESS;
    }
    return FAILURE;
  }
}

// 状态机
class EnemyState {
  update() {
    switch(this.state) {
      case 'PATROL': this.patrol(); break;
      case 'ATTACK': this.attack(); break;
      case 'DODGE': this.dodge(); break;
      case 'DEFEND_BASE': this.defendBase(); break;
    }
  }
}

## ✅ 验收标准

### 功能验收
- [ ] 玩家坦克可流畅控制（上下左右 + 射击）
- [ ] 子弹可击毁敌方坦克和砖块
- [ ] 敌方坦克有 AI 行为（移动 + 射击 + 追踪）
- [ ] 碰撞检测准确（无穿模/漏检测）
- [ ] 基地被摧毁游戏结束
- [ ] 击毁敌机得分正确
- [ ] 游戏状态切换正常（开始/暂停/结束）

### 性能验收
- [ ] FPS ≥ 60（平均）
- [ ] FPS ≥ 45（最低，激烈战斗时）
- [ ] 内存使用 < 100MB
- [ ] 内存增长稳定（无泄漏）
- [ ] 实体数量：子弹≤50，敌机≤4

### AI 验收
- [ ] 敌方坦克会主动攻击玩家
- [ ] 敌方坦克会躲避玩家子弹
- [ ] 敌方坦克会回防基地
- [ ] 不同难度 AI 行为有差异

### 质量验收
- [ ] 代码通过 ESLint 检查
- [ ] 无控制台报错
- [ ] 游戏可连续运行 5 分钟不崩溃
- [ ] 移动端适配（响应式）
