## 📝 技术细节

### 后台运行

```bash
# 方式 1：简单后台
python3 -m http.server 8082 --bind 0.0.0.0 &

# 方式 2：nohup 持久化
nohup python3 -m http.server 8082 --bind 0.0.0.0 > /tmp/game.log 2>&1 &
```

### 检查服务器状态

```bash
# 查看进程
ps aux | grep "http.server"

# 查看端口
netstat -tlnp | grep [端口]

# 查看日志
tail -f /tmp/game.log
```

### 停止服务器

```bash
# 找到 PID
ps aux | grep "http.server"

# 杀死进程
kill [PID]
```
